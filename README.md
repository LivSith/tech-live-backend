# tech-live-backend

Desenvolvido com Python 3.8

## - API de teste(local)
Feita para enteder a estrutura de uma API (com Flask)
### Como rodar:
```
$ cd tech-talk-api
$ virtualenv flask
$ flask/bin/pip install flask
$chmod a+x app.py
$ ./app.py
```
